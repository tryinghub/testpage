// Command Reference

Commands

📦

Collection

5 commands ▼
cl add <pokemon>
Add one or more Pokémon to your collection. Use commas to add multiple at once.
aliases: collection add
cl remove <pokemon>
Remove a Pokémon from your collection.
aliases: collection remove
cl list
View your full collection in a paginated embed.
aliases: collection list
cl raw
View your collection as plain text (or .txt file if large).
aliases: collection raw
cl clear
⚠️ Clear your entire collection.
aliases: collection clear
✨

Shiny Hunt

3 commands ▼
sh
Check your current shiny hunt target.
aliases: hunt, shinyhunt
sh <pokemon>
Start hunting a specific Pokémon. Get pinged when it spawns.
aliases: hunt <pokemon>
sh clear
Stop your current shiny hunt.
also: sh none, sh stop
🔷

Type & Region Pings

4 commands ▼
tp
Open the interactive type pings menu to toggle all 18 types.
aliases: typepings, typeping
tp <types>
Directly toggle one or more types. Example: tp bug grass fire
rp
Open the interactive region pings menu (Kanto → Paldea).
aliases: regionpings, regionping
rp <regions>
Directly toggle one or more regions. Example: rp kanto johto
🔥

Incense

6 commands ▼
inc toggle
Enable or disable the incense watcher for this server.
inc cat add <names>
Add one or more channel categories to monitor for incense purchases.
inc cat remove <name>
Stop monitoring a specific category.
inc cat list
View all monitored categories and their channel counts.
inc pause [all]
Pause incense for this channel, or all monitored categories at once.
aliases: inc p
inc resume [all]
Resume incense for this channel, or all paused channels at once.
aliases: inc r
⭐

Starboard

9 commands ▼
starboard-settings
View current starboard channel configuration.
starboard-all #ch
Log all catches, hatches, and unboxes to a channel.
starboard-shiny #ch
Log shiny catches, hatches, and unboxes.
starboard-gigantamax #ch
Log Gigantamax Pokémon.
starboard-highiv #ch
Log high IV catches (≥90%).
starboard-lowiv #ch
Log low IV catches (≤10%).
starboard-missingno #ch
Log MissingNo catches.
starboard-milestone #ch
Log milestone catches (e.g. 1000th catch).
catchcheck / eggcheck
Manually trigger a starboard check by replying to a message.
⚙️

Settings

6 commands ▼
afk
Toggle pings on/off with interactive buttons — ShinyHunt, Collection, TypePings, RegionPings.
aliases: away
server-settings
View all current server settings.
aliases: ss, ssettings
rare-role @role
Set a role to ping for Legendary/Mythical/Ultra Beast spawns.
regional-role @role
Set a role to ping for regional Pokémon spawns.
only-pings [true|false]
Only send prediction messages when there are pings to deliver.
aliases: op, onlypings
clear-pings [@user]
Clear all ping data for a user or the entire server.
aliases: clearpings, resetpings

Helpful Tools

Breeding

3 commands ▼
breed [count]
Generate optimal breeding pairs from your inventory (max 2 pairs).
iwant "pokemon" move1, move2
Find the breeding chain to teach specific egg moves to a Pokémon.
aliases: chainbreed, cb
canlearn move1, move2
Find all Pokémon that can naturally learn a set of moves.
aliases: wholearns, wl

Inventory

11 commands ▼
add [message_ids]
Add Pokémon to your normal inventory.
addtripmax [ids]
Add to TripMax inventory (high IV breeding).
addtripzero [ids]
Add to TripZero inventory (low IV breeding).
addduel [ids]
Add to Duel inventory for egg move breeding.
aliases: ad
remove [ids] [--category]
Remove Pokémon from a specific inventory category.
aliases: rm | flags: --normal --tripmax --tripzero --duel
releaseall --n <name>
Release all Pokémon matching name and category filters.
aliases: ra
clear <category>
Clear an entire inventory category. Options: inv, tripmax, tripzero, duel, all.
inventory [filters]
View normal inventory. Filter by gender, name, type, region, cooldown.
aliases: inv, invnormal
invtripmax [filters]
View TripMax inventory with optional filters.
aliases: trip31, tripmax
invtripzero [filters]
View TripZero inventory with optional filters.
aliases: tripzero, trip0
stats
View inventory statistics across all categories.

shinystats

Settings

6 commands ▼
settings
Open the interactive settings menu with all breeding configurations.
aliases: gear, sg
mode [selective|notselective]
Set breeding mode. Selective pairs old IDs (≤271800) with new IDs (≥271900).

Set breeding target. Options: all, mychoice, tripmax, tripzero, gigantamax, regionals, or Pokémon names.
setmale / setfemale
Set specific males/females for MyChoice breeding target.

Manage Pokémon cooldown list for breeding.
aliases: cd add / cd remove / cd list / cd clear
setnickname <id> <nick>
Set a nickname for a specific Pokémon.
aliases: nick
